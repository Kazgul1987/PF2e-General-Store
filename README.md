# PF2e General Store

Ein leichtgewichtiges Foundry-VTT-Modul für **Pathfinder 2e**, das eine "General Store"-Ansicht bereitstellt. Ziel ist eine schnelle Auswahl und Filterung gängiger Einträge wie **Items** und **Spells** direkt im Dialog.

## Features
- Filter für Item- und Spell-Einträge
- Volltextsuche innerhalb der angezeigten Einträge
- Deutsche Lokalisierung (DE)
- Kompakte, übersichtliche Dialog-Oberfläche

## Voraussetzungen
- Foundry VTT v13 (getestet)
- PF2e-System v7.9.1 (getestet)

## Installation
1. Lade das Modul herunter oder installiere es über die Foundry-Paketverwaltung.
2. Aktiviere **PF2e General Store** in den Modul-Einstellungen deines Worlds.

## Nutzung
Öffne den General-Store-Dialog und nutze die Filter sowie die Suchleiste, um schnell die gewünschten Einträge zu finden.
